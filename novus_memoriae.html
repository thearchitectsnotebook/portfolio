Novus Memoriae

redefining the C++ pointer

Explore Novus Memoriae

Overview

Novus Memoriae is a library that redefines the C++ pointer into an entirely new class. The Ptr class operates in the same way as the original C++ pointer (besides some syntax differences). However, instead of within the RAM, all memory is saved in a byte array, within the Memory class; this makes memory portable.
